Who doesn't like a pretty map? PrettyMaps was developed by [marceloprates](https://github.com/marceloprates/prettymaps) and is an easy-to-use 
map-styler using OpenStreetMap data. The tool allows you to build maps from OSM feature tags
and style them using various preset json files. 

The following are school campuses that I've either visited or attended personally. I customized the coloring of buildings to represent the school's official colors. It takes time to familiarize yourself with all the various OSM tags and most may not be applicable to each location. I spent a lot of time researching each location's features in QGIS to ensure I was rendering the most prevalent structures and roads. Many college campuses have great networks of footpaths and trails that I wanted to make sure I included as well. Overall, the tool is quite easy to use and creates fun and creative looking maps. Perfect for gifts!
